## 1 设计概述

### 1.1 设计目标

基于需求文档，本UI交互规范旨在实现：
- 表格纵向滚动时表头固定在顶部保持可见
- 表头与内容列精确对齐
- 支持深色/浅色主题无缝切换
- 响应式布局适配多种屏幕尺寸
- 提供流畅的用户交互体验

### 1.2 设计原则

| 原则 | 说明 |
|-----|------|
| 一致性 | 与现有系统UI风格保持一致 |
| 清晰性 | 表头信息清晰可读 |
| 流畅性 | 滚动和固定切换平滑无卡顿 |
| 兼容性 | 支持深色/浅色主题及多种屏幕尺寸 |

## 3 滚动交互设计

### 3.1 滚动监听机制

#### 3.1.1 触发条件

| 条件 | 行为 |
|-----|------|
| 内容高度 ≤ 容器高度 | 不启用固定表头，正常显示 |
| 内容高度 > 容器高度 | 启用固定表头功能 |

#### 3.1.2 滚动状态判断

```
滚动位置 > 表头偏移量 → 固定表头显示
滚动位置 ≤ 表头偏移量 → 表头恢复原位
```

### 3.2 表头同步机制

#### 3.2.1 克隆表头创建

| 步骤 | 操作 |
|-----|------|
| 1 | 检测表格是否需要固定表头 |
| 2 | 克隆原始表头DOM |
| 3 | 为克隆表头添加固定定位样式 |
| 4 | 同步列宽度 |
| 5 | 插入到容器顶部 |

#### 3.2.2 同步更新策略

| 事件 | 同步操作 |
|-----|---------|
| 滚动事件 | 保持表头位置固定 |
| 窗口resize | 重新计算列宽度 |
| 主题切换 | 更新表头样式 |
| 数据更新 | 重新克隆表头 |

### 3.3 过渡动画

#### 3.3.1 固定/取消固定过渡

| 属性 | 浅色主题 | 深色主题 |
|-----|---------|---------|
| 过渡属性 | `box-shadow, opacity` | `box-shadow, opacity` |
| 过渡时长 | `200ms` | `200ms` |
| 过渡函数 | `ease-in-out` | `ease-in-out` |

#### 3.3.2 状态切换效果

| 状态 | 效果 |
|-----|------|
| 开始固定 | 阴影渐显，透明度保持100% |
| 取消固定 | 阴影渐隐 |

## 4 列对齐设计

### 4.1 宽度计算机制

#### 4.1.1 初始宽度计算

| 步骤 | 操作 |
|-----|------|
| 1 | 表格渲染完成后 |
| 2 | 获取内容区域各列实际宽度 |
| 3 | 存储列宽度数组 |
| 4 | 设置克隆表头各列宽度 |

#### 4.1.2 宽度来源优先级

| 优先级 | 来源 | 说明 |
|-----|-----|------|
| 1 | 显式设置 | 用户指定的列宽度 |
| 2 | 内容自适应 | 根据单元格内容自动计算 |
| 3 | 均分 | 剩余空间平均分配 |

### 4.2 动态同步机制

#### 4.2.1 同步时机

| 时机 | 触发条件 |
|-----|---------|
| 初始同步 | 表格首次渲染完成 |
| 窗口resize | 窗口大小变化结束后 |
| 数据更新 | 查询结果数据变更 |
| 列调整 | 用户调整列宽度 |

#### 4.2.2 同步流程

```
事件触发
    ↓
获取内容列宽度
    ↓
更新表头列宽度
    ↓
触发重绘
```

### 4.3 响应式适配

#### 4.3.1 断点设置

| 断点 | 屏幕宽度 | 适配策略 |
|-----|---------|---------|
| 移动端 | < 768px | 允许横向滚动，表头跟随横向滚动 |
| 平板端 | 768px - 1024px | 保持列对齐，适当缩小内边距 |
| 桌面端 | ≥ 1024px | 正常显示 |

#### 4.3.2 最小宽度限制

| 元素 | 最小宽度 |
|-----|---------|
| 表格容器 | `360px` |
| 表格列 | `80px` |
| 表头单元格 | `80px` |

## 9 响应式设计规范

### 9.1 断点配置

| 断点名称 | 屏幕宽度 | 布局策略 |
|---------|---------|---------|
| xs | < 480px | 单列显示，允许横向滚动 |
| sm | 480px - 768px | 紧凑布局，减少内边距 |
| md | 768px - 1024px | 标准布局 |

| 断点 | 表头高度 | 单元格内边距 | 字体大小 |

### 9.3 横向滚动处理

#### 9.3.1 触发条件

## 10 前端开发实现说明

### 10.1 核心技术点

#### 10.1.1 滚动监听

使用 `useEffect` 监听滚动事件，结合 `Ref` 获取DOM元素。

#### 10.1.2 宽度同步

使用 `ResizeObserver` 监听表格宽度变化，动态同步列宽度。

#### 10.1.3 主题切换

使用CSS变量或条件类名实现主题切换。

### 10.2 性能优化建议

#### 10.2.1 节流处理

对滚动事件进行节流，避免频繁重绘。

#### 10.2.2 虚拟滚动（可选）

对于超大数据量，可集成虚拟滚动库。

#### 10.2.3 CSS优化

使用 `will-change` 提示浏览器优化渲染：
```css
.fixed-header {
  will-change: position, box-shadow;
}
```
