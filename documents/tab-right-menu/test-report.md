# 标签页右键菜单功能测试报告

## 1. 测试概述

### 项目背景
本项目为 SQL 查询编辑器页面，需要在标签页名称区域添加鼠标右键菜单功能，支持批量关闭标签页操作。

### 测试范围
- 右键菜单的显示与隐藏
- 三个菜单选项的功能正确性：删除除本页外的其他标签、删除左边的标签、删除右边的标签
- 业务规则验证：标签页数量判断、禁用状态控制

### 测试类型
- 功能测试
- 边界测试
- 异常测试

## 3. 业务流程梳理

```
用户右键点击标签页名称
        │
        ▼
┌──────────────────────────────┐
│ 判断标签页数量是否 > 1       │
└──────────────────────────────┘
        │
        ├── 否 → 不显示右键菜单
        │
        ▼ 是
┌──────────────────────────────┐
│ 根据当前标签位置计算禁用状态 │
│ - 第一个标签: "删除左边"禁用 │
│ - 最后一个标签: "删除右边"禁用 │
└──────────────────────────────┘
        │
        ▼
┌──────────────────────────────┐
│ 显示右键菜单（3个选项）      │
└──────────────────────────────┘
        │
        ├─ 点击"删除除本页外的其他标签" → closeOtherTabs()
        ├─ 点击"删除左边的标签" → closeLeftTabs()
        └─ 点击"删除右边的标签" → closeRightTabs()
```

## 6. 边界值 & 异常场景专项测试

| 测试场景 | 前置条件 | 操作步骤 | 预期结果 | 优先级 |
| :--- | :--- | :--- | :--- | :--- |
| 边界：2个标签页删除其他 | 存在2个标签页（标签A、标签B），当前选中标签A | 右键点击标签A → 选择"删除除本页外的其他标签" | 仅保留标签A，标签B被删除 | 高 |
| 边界：2个标签页删除左边 | 存在2个标签页（标签A、标签B），当前选中标签B | 右键点击标签B → 选择"删除左边的标签" | 仅保留标签B，标签A被删除 | 高 |
| 边界：2个标签页删除右边 | 存在2个标签页（标签A、标签B），当前选中标签A | 右键点击标签A → 选择"删除右边的标签" | 仅保留标签A，标签B被删除 | 高 |
| 边界：第一个标签删除左边（禁用） | 存在3个标签页，当前选中第一个标签 | 右键点击第一个标签 | "删除左边的标签"选项禁用 | 高 |
| 边界：最后一个标签删除右边（禁用） | 存在3个标签页，当前选中最后一个标签 | 右键点击最后一个标签 | "删除右边的标签"选项禁用 | 高 |
| 异常：快速连续右键点击 | 存在多个标签页 | 快速连续右键点击不同标签 | 每次点击都正确显示菜单，无异常 | 中 |
| 异常：菜单位置超出视窗 | 存在多个标签页，鼠标在屏幕边缘 | 在屏幕边缘右键点击标签 | 菜单自动调整位置，完整显示在视窗内 | 中 |

## 9. 潜在风险与 Bug 预判

| 风险点 | 描述 | 严重程度 | 建议 |
| :--- | :--- | :--- | :--- |
| 数组越界 | closeLeftTabs/closeRightTabs 可能在边界条件下出现数组越界 | 高 | 增加数组长度检查，确保索引有效 |
| 状态不同步 | 删除标签后，activeTabId 可能指向已删除的标签 | 高 | 删除后确保 activeTabId 更新为有效标签 |
| 菜单位置溢出 | 菜单在屏幕边缘可能被截断 | 中 | 实现位置自动调整逻辑（已实现） |
| 并发操作问题 | 快速连续操作可能导致状态不一致 | 中 | 确保状态更新使用函数式 setState |
| 禁用状态判断错误 | 禁用状态计算错误导致功能异常 | 高 | 增加单元测试覆盖边界条件 |

## 10. 测试结论与上线建议

### 测试结论
根据需求文档和代码实现分析，右键菜单功能已基本实现，但需完成以下测试验证：

1. **必须通过的测试项**：
   - 标签页数量判断（≤1不显示菜单）
   - 三个菜单选项功能正确性
   - 禁用状态控制（第一个/最后一个标签）
   - 菜单关闭机制（点击外部、ESC键）

2. **建议补充的测试**：
   - 单元测试：对 closeOtherTabs、closeLeftTabs、closeRightTabs 函数进行单元测试
   - 集成测试：验证菜单与标签页状态的交互

### 上线建议
1. 在测试环境完成所有功能测试用例验证
2. 确保无严重缺陷（致命/严重级别）存在
3. 完成浏览器兼容性测试
4. 建议增加自动化测试覆盖核心功能
